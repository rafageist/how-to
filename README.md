# How-To Knowledge Base

Public repository for practical Markdown documentation.

This repository stores reusable guides, commands, scripts, troubleshooting notes, and operational procedures.

## Structure

- `docs/`: finalized guides by domain.
- `scripts/`: reusable helper scripts.
- `indexes/`: navigation by domain, tool, and problem.
- `_templates/`: document templates.
